Project Title - Adress Linkage Key!!!

Project Description:

Basic Project Info: Utilizing an algorithm provided by the United States Postal Service (USPS), create an address linkage key similar to the CASS delivery point. The solution provided will be scaled to run against full United States population datasets using Big Data technology.
Vision Of Project: Develop a tool or set of tools to compute the delivery point of a given address. Initially the tool(s) will be tested against properly formatted addresses, further iterations of the tool(s) will be tested against improperly formatted addresses as well as being used to determine if the dataset being examined contains certain types of delivery points such as High-rises.
Tools & Technologies Utilized: Python,Git and spark
